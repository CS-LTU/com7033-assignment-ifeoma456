
# Healthcare Dataset - Stroke Prediction

## Overview
This dataset contains healthcare information used for stroke prediction and analysis. It includes patient demographics, medical history, and health metrics to help identify risk factors associated with stroke occurrence.

## Dataset Contents
- **Patient Demographics**: Age, gender, residence type
- **Medical History**: Hypertension, heart disease, smoking status
- **Health Metrics**: Average glucose level, BMI
- **Target Variable**: Stroke occurrence (binary classification)

## Features
| Feature | Description |
|---------|-------------|
| `age` | Patient age in years |
| `gender` | Male, Female, Other |
| `hypertension` | 0 = No, 1 = Yes |
| `heart_disease` | 0 = No, 1 = Yes |
| `avg_glucose_level` | Average glucose level in blood |
| `bmi` | Body Mass Index |
| `smoking_status` | Never, formerly, smokes, Unknown |
| `stroke` | 0 = No stroke, 1 = Stroke |

## Usage
This dataset is useful for:
- Building machine learning models for stroke prediction
- Analyzing stroke risk factors
- Healthcare research and analysis

## Size
Approximately 5,000+ patient records with 11 features.
