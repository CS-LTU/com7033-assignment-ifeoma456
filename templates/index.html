About This System

According to the World Health Organization (WHO), stroke is the second leading cause of death globally. This system is designed to help healthcare professionals manage patient data covering demographics, medical history, and lifestyle factors to predict stroke likelihood and assist in preventive healthcare.

Security Features: Password hashing, CSRF protection, input validation, XSS prevention, secure session handling, and comprehensive error logging.
